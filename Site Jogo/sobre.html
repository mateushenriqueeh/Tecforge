Sobre o Desenvolvimento do Jogo


Plataforma de desenvolvimento: Esse jogo foi desenvolvido utilizando a Engine Contruct 3, que oferece uma interface intuitiva e poderosas ferramentas de desenvolvimento para criadores de jogos de todos os níveis de habilidade



Elementos Gráficos: As sprites utilizadas nesse jogo foram retiradas do Craftpix, uma plataforma popular para recursos gráficos de alta qualidade.


Perspectiva Top-Down: Os jogadores experimentam o labirinto em uma perspectiva top-down, o que oferece uma visão panorâmica do ambiente e adiciona uma camada de estratégia à jogabilidade


Objetivo do Jogo: O objetivo principal de "As trilhas do Labirinto" é guiar o personagem principal através do labirinto, enfrentando os inimigos encontrados pelo caminho. Os jogadores devem usar suas habilidades para derrotar os inimigos e encontrar o caminho para fora do labirinto e assim alcançar a vitória.
